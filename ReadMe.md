 ### initialize git
`git init`


### make some changes



### add those files to staging
  `git add <fileName>`


### commit those changes
 `git commit -m 'Commit message'`


### create a branch
  `git branch <branch_name>`


### list all the branches

  `git branch`

   or 

  `git branch -a`


### Switch to a new branch
 `git checkout <branch_name>`
